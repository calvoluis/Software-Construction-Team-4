# Software-Construction-Team-4
# READ ME

In order to use this project the following files must be downloaded and added to the project as referenced libraries:
  JavaParser - https://github.com/javaparser/javaparser attach the javaparser-core-2.4.0.jar
  XMLStream - http://x-stream.github.io/download.html xmlpull-1.1.3.1.jar xp33_min-1.1.4c.jar xstream-1.4.9.jar
  Grapviz - http://www.graphviz.org/Download.php
  
The CFGMaker.java both have main statements that allow them to execute seperatly.

To run CFGMaker input the pathfile for a java file you wish to read (Ex. C:\Users\LuisCalvo\Downloads\Soft. Const\Software-Construction-Team-4-master\OverlappingRectangles.java). This will result in the program printing the nodes and edges as well as saving the file in the CFGMaker folder as the specified name and path.
